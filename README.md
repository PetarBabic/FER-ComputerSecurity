# FER - Computer Security

Lab solutions for Computer Secutiry during the academic year 24/25

Lab 1 - Design and implement a simple and secure password manager prototype using symmetric cryptography\
Lab 2 - This lab focuses on identifying and exploiting buffer overflow vulnerabilities in the zookws web server via HTTP requests to inject and execute code.\
Lab 3 - Study web application vulnerabilities with the goal of learning how to test and secure web applications in the future\
&emsp; **Command Injection**\
&emsp; **SQL Injection**\
&emsp; **XSS (Cross Site Scripting)**\
&emsp; **File Inclusion**\
Lab 4 - This lab covers discovering network services and configuring a Linux NetFilter/iptables firewall to secure a network.
